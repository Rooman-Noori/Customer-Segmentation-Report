# Customer Segmentation Project

## Objective

Segment customers using K-Means clustering based on income and spending.

## Dataset

Mall Customers Dataset (200 records)

## Tools

-   Python
-   Pandas
-   Scikit-learn
-   Matplotlib

## Method

-   Data Cleaning
-   Feature Selection
-   Standard Scaling
-   Elbow Method
-   K-Means Clustering

## Result

Five customer segments identified: - Premium Customers - Regular
Customers - Budget Customers - Potential Customers - Impulsive Buyers

## Business Insights

Targeted marketing strategies can be applied to each group.
